fazendo coisas com git para o meu primeiro repositorio, fazendo o git clone, push comit e o readme


## como enviar projetos para o github

- precisa ter o **git, github desktop** instalado
- abrir o **cmd** na pasta do projeto e executar o comando `git init `
- fazer o **comit** no github desktop e depois clicar em `push` para enviar
    - se é a primeira vez precisa clicar em `publish`
    - lembrar de preferencialmente deixar o projeto **publico**

## como baixar projetos

abrir o **cmd**e digitar `git clone  [link do repositorio]`
